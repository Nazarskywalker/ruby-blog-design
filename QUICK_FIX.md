# ШВИДКЕ ВИПРАВЛЕННЯ ДОСТУПУ З ТЕЛЕФОНУ

## 🚨 ПРОБЛЕМА: Сервер працює на ПК але не доступний з телефону

## ✅ РІШЕННЯ (по черзі):

### 1. ЕКСТРЕНИЙ СПОСІБ (100% працює):
```bash
ruby emergency_fix.rb
ruby emergency_server.rb
```
Потім на телефоні відкривайте адреси з екрану

### 2. ПОРТ 8080 (найкращий):
```bash
ruby start_8080.rb
```
На телефоні: http://192.168.x.x:8080

### 3. ВИМКНЕННЯ БРАНДМАУЕРА:
1. Win + R → firewall.cpl
2. Увімкнення/вимкнення брандмауера
3. Вимкнути для всіх мереж
4. Перезапустити сервер

### 4. ПЕРЕВІРКА WI-FI:
- Телефон і ПК в одній мережі?
- IP-адреси однакові (напр. 192.168.1.x)?
- Спробуйте інший Wi-Fi

### 5. АНТИВІРУС:
Вимкніть антивірус на 5 хвилин і спробуйте знову

## 🔍 ДІАГНОСТИКА:
```bash
ruby test_all_ports.rb
```
Покаже які порти працюють

## 📱 ЯКІ АДРЕСИ ПРОБУВАТИ:
Після запуску сервера на телефоні пробуйте всі:
- http://192.168.56.1:8080
- http://192.168.50.86:8080  
- http://172.20.96.1:8080

## 💡 ГОЛОВНА РЕКОМЕНДАЦІЯ:
Спочатку `ruby emergency_fix.rb` - це створить мінімальний сервер,
який точно працюватиме з телефону!
