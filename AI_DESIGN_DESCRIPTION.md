# AI-дизайн головної сторінки блогу

## Що було згенеровано

Для домашнього завдання створено сучасний дизайн головної сторінки блогу у стилі AI-generated landing page.

Сторінка містить:

- великий hero-блок із коротким описом блогу;
- кнопки швидкої дії для створення та перегляду постів;
- блок статистики блогу;
- картки категорій;
- секцію останніх публікацій;
- виділений останній пост;
- адаптивну сітку карток для інших постів;
- окремий empty state, якщо постів ще немає;
- адаптивну верстку для мобільних пристроїв.

## Де реалізовано

- `views/index.erb` — HTML/ERB-структура головної сторінки.
- `public/style.css` — CSS-стилі для нового дизайну.
- `app.rb` — додано `require 'json'` для коректної роботи JSON-відповіді `/check-updates`.

## Особливості дизайну

Дизайн використовує:

- сучасні картки з тінями;
- градієнтні акценти;
- адаптивну CSS Grid-верстку;
- великі заголовки;
- закруглені блоки;
- clean UI у стилі сучасного блогу.
